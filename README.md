## Getting Started
### Install
- Clone the repository
  ```
  git clone https://github.com/exploreka/backend-api.git
  ```
- Install NPM modules
  ```
  npm install
  ```

### Run
- Create `.env` file and setup with [the following variables](env.example)
- Run the app
  ```
  npm run start
  ```
